ProSports is an advanced all-in-one sports plugin that helps sports clubs to set up and manage a sports league or club website quickly and easy.

== Description ==

= Create a Professional Sports Website =
Transform your WordPress site into a fully configurable sports team, club, or league website. Features a suite of advanced sports tools including fixtures, results, automated standings, players rankings, and individual profiles for players, clubs, staff and more.

== Installation ==
= The Minimum Requirements =
* WordPress 3.8 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Automatic Installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t even need to leave your web browser. To do an automatic install of ProSports, log in to your WordPress admin panel, navigate to the Plugins menu and click Add New.

In the search field type “Pro Sports” and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking Install Now. After clicking that link you will be asked if you’re sure you want to install the plugin. Click yes and WordPress will automatically complete the installation.

= Manual Installation =

The manual installation involves downloading our sports plugin and uploading it to your server via your FTP application.

1. Download the ProSports plugin file to your computer and unzip it
2. Using an FTP program, or your hosting panel, upload the unzipped plugin folder to your WordPress installation’s wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu within the WordPress admin.

== Frequently Asked Questions ==
= Which sports does this plugin support? =

The plugin will support most team sports with a scoring system. You can customize the table columns and player statistics via ProSports settings. We\'ve included presets for commonly requested sports, and you can also add your own.

= Will ProSports work with my theme? =

Yes; ProSports will work with any theme, but may require some styling to make it match nicely.

= Where can I report bugs or contribute to the project? =

Bugs can be reported on [ProSports GitHub repository](https://github.com/ProSports/ProSports/issues) or by [contacting us here] (http://livescores.website/prosports/).

= Can I import teams/players/events from a CSV file? =

Yes, CSV importers are included. Go to Tools > Import and choose from one of the ProSports CSV importers.
